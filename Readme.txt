Instruções de uso do jogo:
Para executar o jogo apenas compilar o main utilizando Eclipse IDE ou IntelliJ IDEA

Atalhos: 
W A S D - mover o personagem;
P - atacar enquanto se move na direção do inimigo;
E - Pegar moedas.


Game Usage Instructions:
To run the game, simply compile the main file using Eclipse IDE or IntelliJ IDEA.

W A S D – Move the character
P – Attack while moving toward the enemy
E – Collect coins
